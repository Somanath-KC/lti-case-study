**Environment Setup!**
1. Clone this repo
	 ```
	 $ git clone https://github.com/Somanath-KC/lti-case-study.git
	 ```
 
2. Get into project directory
	 ```
	 $ cd lit-case-study
	 ```
3. Run the Application
	 ```
	 $ python sms.py
	 ```
